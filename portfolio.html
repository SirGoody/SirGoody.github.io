Base Conversion Learning Tool

This project here was a nine week project I had with a group of 4 people. We decided to do this because we thought it could also help classmates who are taking a class that involves a lot of these conversions. My main contributions to this was creating the GUI, button-on-action events, scene switching, and creating the tutorials. The top picture is of the main menu. I had to use photoshop to create the images that you see running across the side. The bottom photo shows the Quiz Mode (left), Base Calculator (top right), and the Conversion Tutorials (bottom right).
