MacBeth Adventure Text-based Game

I created this game as a final project in my Java Programming class. This game can be ran by either using a Java IDE (I use IntelliJ developed by JetBrains), or you can use the command line (./game.java). In this game you basically make your way through the cave, and find all of the items for the witch to make her special concoction to get you out of that place. You can type help in the game to see all of the commands, I also specify that within the game itself. This game is pretty fun and it was really fun to make. This shows off my object oriented programming skills, each thing you pick up is an object that is relative to the room it is stored in.

Conway's Game of Life

I recreated this game as a way to show my skills with 2D Maps in Java. This game only runs off of a command line or an IDE, you can look at it in my GitHub repositories to get the code if you want to. Basically Conways game of life means that each cell with 0 or 1 neighbors dies from solitude. Also if it has 4 or more neighbors it dies from overpopulation. The cell can only live if it has 3 or 4 neighbors. I tried to recreate that using a Map in java and I had to access the neighbors and see their positions. I went off of the rules that I stated earlier.

Grading Algorithm

I made this algorithm using Eclipse based off of a Professor grading scale. This would help me see where myself, and anyone else, was at in that class. This shows that I can make medium-easy level algorithms with a breeze. This is the puzzle that I love to figure out. His grading scale was ((totalpoints/2) + (labpoints/2)) = C cut off. From that we then gather the highest grade in the class, and then look at the difference between the highest grade and the C cut off and make leveled increments to get the other grades in the grading scale. You can see my code for more information on the algorithm. You can only get an A, B, C, and an F.
